# External Services Manifest

This document tracks third-party services and required accounts for the MenuMaker remediation.

## AWS (Amazon Web Services)
- **What it does in this project:** Hosts backend, database, and manages secrets.
- **How to get credentials:** Create an IAM user and generate Access Keys in the AWS Console.

## Apple Developer Program
- **What it does in this project:** Required for signing and deploying iOS apps.

## Google Play Console
- **What it does in this project:** Required for deploying Android apps.

## Stripe
- **What it does in this project:** Handles payments and payouts.

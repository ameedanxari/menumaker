# 2026 documentation archive

This folder is the approved destination for superseded 2026 root/spec/platform reports after `scripts/docs/apply_disposition.py --apply` is run from an approved inventory.

Archived files must include provenance frontmatter with:

- `archived_at`
- `original_path`
- `original_sha256`
- `superseded_by`
- `retention_reason`

The inventory at [../../governance/document-inventory.csv](../../governance/document-inventory.csv) is the source of truth for proposed moves, replacements, and retention reasons.
